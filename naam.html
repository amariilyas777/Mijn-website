Project: Homepagina Website

Beschrijving

Dit project is een moderne homepagina gebouwd met HTML en CSS. De pagina heeft een header, banner, navigatiemenu, afbeeldingensectie en footer.
Het ontwerp is responsief en volgt het kleurpalet van de website (blauw en paars).

Gebruikte technologieën

  • HTML5
  • CSS3
  • Flexbox
  • Responsive design

Wat heb ik geleerd?

Ik leerde werken met lay-outstructuren, flexbox, semantische HTML en het creëren van componenten binnen een website. Dit project vormt de basis voor mijn volledige portfolio.
